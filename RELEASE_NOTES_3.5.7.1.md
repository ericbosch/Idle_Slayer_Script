# Idle Runner v3.5.7.1

Patch release following 3.5.7.

## Changes in 3.5.7.1

- **Boss Fight Victor** — Removed leftover PrintScreen (Alt+PrintScreen) call during flame attack.
- **Build script** — Dropped unsupported `/verbose` flag; wrapper can be run via AutoIt3 when only `.au3` wrapper exists.
- **Version** — UI version set to 3.5.7.1.

## Requirements

- Windows (Steam version of Idle Slayer)
- Game: 1280×720, windowed, English, focus for full automation
- Keybinds: jump = Up arrow, boost = Right arrow
