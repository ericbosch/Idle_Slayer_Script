# Idle Runner v3.5.7

Release version **3.5.7** — aligned with upstream [Devil4ngle/Idle_Slayer_Script](https://github.com/Devil4ngle/Idle_Slayer_Script) plus fork enhancements.

## Changes in 3.5.7

- **PixelSearch step parameter** — Added step `1` to `PixelSearch` calls across the script for more reliable UI detection (less sensitive to exact pixel match).
- **Version** — UI version set to 3.5.7.

## Requirements

- Windows (Steam version of Idle Slayer)
- Game: 1280×720, windowed, English, focus for full automation
- Keybinds: jump = Up arrow, boost = Right arrow

## Fork extras (this repo)

This build includes all upstream 3.5.7 behaviour plus:

- Armory / Sell Armory (Excellent / Non-Excellent / Victor Ring options)
- Boss Knight vs Victor detection
- SellArmory, ClaimQuests, and related flow improvements
- Optional tweaks (e.g. silver box color)

See [README](https://github.com/ericbosch/Idle_Slayer_Script#readme) for full setup and options.
